효과적인 데이터 입력 방법

✅ FAQ 형태 추천 — "Q: 환불은 어떻게 하나요? A: 결제일로부터 7일 이내..." 형태가 검색에 가장 효과적입니다.

✅ 주제별 문서 분리 — 하나의 파일에 모든 내용을 넣지 마세요. 요금제, 환불, 배송 등 주제별로 나눠서 올리면 정확도가 높아집니다.

✅ 자연스러운 문장 사용 — "36250, monthly" ❌ → "Starter 플랜의 월간 요금은 36,250원입니다" ⭕

✅ 적절한 문서 길이 — 200~5,000자가 적당합니다. 너무 짧거나 길면 검색 품질이 떨어집니다.

ℹ️ SQL/JSON 파일은 자동으로 자연어 변환됩니다. 파일명 제목은 검색에 유리한 제목으로 자동 변경됩니다. 중복 문서는 자동 감지됩니다.

💡 FAQ, 정책 문서, 안내문 등을 직접 입력하세요. 질문-답변 형태가 가장 효과적입니다.

💡 SQL, JSON, CSV 파일은 자동으로 자연어로 변환됩니다. 여러 파일을 동시에 업로드할 수 있습니다.
